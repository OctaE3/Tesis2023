Instituto CTC

Rosario, Colonia, Uruguay — 2023

Autores:
Alex Besozzi, Octavio Etchevarren

Resumen:
Este repositorio contiene el código fuente y recursos del proyecto de tesis "La Marcelina", desarrollado en 2023 como parte de la carrera de Analista Programador. El proyecto consiste en un sistema de software con componentes backend y frontend.

Se implementó un prototipo funcional que integra tecnologías web modernas para cubrir las siguientes necesidades:
1. Controlar los tiempos de elaboración de los diferentes productos.
2. Mantener un orden en las planillas para evitar pérdidas de tiempo o
información.
3. Simplificar el uso de los identificadores compuestos de los productos.
4. Simplificar el trabajo de búsqueda de datos.
5. Centralizar la información.
6. Mejorar la calidad de la información.
7. Mejorar el acceso a la información.
8. Brindar un acceso cómodo y rápido a la información desde cualquier lugar.

Objetivos principales:

1. Recordar al administrador cuando se deben rellenar las planillas.
2. Optimizar el tiempo de obtención de la información por parte del
administrador.
3. Mejorar el tiempo de búsqueda de datos en un 40%.
4. Automatizar el ingreso de la información al sistema.
5. Disminuir los posibles errores al momento de ingresar la información.
6. Lograr una información más limpia y concisa para evitar posibles errores.
7. Optimizar la disponibilidad de la información.
8. Poder obtener las planillas en papel si es necesario.

Tecnologías y herramientas principales:

- Backend: Java (18.3% del repositorio).
- Frontend: JavaScript (81.6%), Node.js (evidenciado por package-lock.json).
- Otros: Configuraciones de VS Code (.vscode), control de versiones con Git.
